Truth Table
The truth table is generated automatically from the circuit layout. When a value of an input element is changed, the corresponding inputs combination on truth table is highlighted. Also, when you click a row of the truth table, the circuit's inputs are updated with the corresponding combination. Clicking on header cell displays the boolean expression of the output.
